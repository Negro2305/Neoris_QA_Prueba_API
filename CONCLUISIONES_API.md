

# Conclusiones de las pruebas API

- Signup:
  - Permite crear nuevos usuarios correctamente.
  - Intentar crear un usuario existente devuelve un error adecuado.

- Login:
  - Con credenciales correctas devuelve éxito.
  - Con credenciales incorrectas devuelve error 401.

- Logs y capturas permiten evidenciar la respuesta de cada prueba.
- Los reportes HTML permiten una visualización clara de los resultados.
